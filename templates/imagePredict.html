Denta Cad

Periodontal diseases are some of the most common diseases that mankind faces. Periodontitis is the 6th most prevalent disease worldwide. It leads to alveolar bone loss, tooth loss, edentulism, and masticatory dysfunction, which indirectly affects nutrition. Periodontitis also impairs quality of life and self-esteem, which imposes huge socio-economic impacts and healthcare costs. In recent years, Deep Learning (DL) architectures have gained popularity due to their success in different computer vision and pattern recognition tasks. We develop a Convolutional Network model that involves multiple non-linear transformations of the data, with the goal of yielding more abstract and ultimately more useful representations. These methods are becoming increasingly popular, often outperforming traditional approaches that include handcrafted features for data representation and machine learning methods for learning tasks. This has happened in the context of an unprecedented growth of available digital data and powerful computational resources. In DentaCad, we have developed an AI driven software image processing tool to diagnose and monitor Periodontal pathologies using ML and AI techniques on dental images (panoramic radiograph). This is a clinical diagnostic tool to identify and monitor Periodontal Disease using X-Ray images. The initial version is developed using cutting edge research and applied Artificial Intelligent (AI) and deep learning approach. The Machine learning model was used to find the correlation between Panoramic X-Ray images of patients with Periodontal disease. This is a fully automated pipeline for detection of Periodontal disease on X-Ray images. Fully convolutional Neural Network (CNN)was used to detect the teeth, followed by a classifier to predict the Periodontal disease . Currently we developed an initial model for detection of periodontally compromised teeth on intraoral radiographs using convolutional Neural Network.
